## 终端使用技巧

####设置Terminal 代理

运行命令：

 ```Bash
export http_proxy=http://proxyAddr:port
export https_proxy=http://proxyAddr:port
 ```

设置ss代理：

```Bash
export http_proxy="socks5://127.0.0.1:1080"
export https_proxy="socks5://127.0.0.1:1080"
```



使终端通过ss代理所有网络：

```Bash
export ALL_PROXY=socks5://127.0.0.1:1080
```
